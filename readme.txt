Order on WhatsApp for WooCommerce allows your customers to submit their orders through WhatsApp, directly from the WooCommerce product page.

== Description ==

With this plugins you can creates a custom message for redirect your customer to WhatsApp after checkout with all order data.

Also, you can add a button in product page, cart page or use the Elementor Widget for add a button in any page and select what product get with the click.

== Installation ==

1. Upload the plugin files to the WordPress
2. Activate the plugin in WordPress
3. Go to "Order on WhatsApp for WooCommerce" in your WordPress admin panel
4. Enter your phone number and other informations
